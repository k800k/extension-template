# Contributing

Contributions certify the Developer Certificate of Origin with `git commit -s`.

This repository accepts content extensions only. Every new source begins as `approvalRequired`. Before activation, complete its specification, privacy and rights records, host declarations, contract tests, live smoke-test record, and review status.

Run `npm run conformance && npm run check && npm test && npm run bundle` before opening a pull request. Extension versions use Semantic Versioning and commits should follow Conventional Commits.
